## Features
- **Bilingual Documentation**: Support for both Croatian and English.
- **Modern UI**: Interactive sidebar, light/dark mode, and responsive design.
- **AI-Powered**: Integrated "Ask BornAI" buttons for instant help.
- **Beginner Friendly**: Dedicated section for developers new to APIs.
- **Implementation Examples**: Code snippets for various frameworks and scenarios.

## Live Documentation
Check out the live guide at: [https://sudregapi.bornai.app](https://sudregapi.bornai.app)

## Getting Started
To view this guide locally:
1. Clone the repository.
2. Open `index.html` in any modern web browser.
